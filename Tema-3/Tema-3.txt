	Creación Web Phising (AMAZON)

Para crear la web de phising empezaria buscando la web a clonar ( en este caso seria AMAZON ). 
Utilizaría VISUAL STUDIO CODE para empezar escribiendo el codigo de la pagina con HTML para los formularios(Metodo POST) y CSS para 
que visualmente sea parecida a la web de origen, despues incluiria PHP para recibir los datos de los formularios (Metodo POST) y 
enviarmelos a mi base de datos MYSQL, despues usando PHP redirigiría a las victimas a la pagina original de AMAZON.

envio masivo de correos con PYTON, por ejemplo con ESTE REPOSITORIO de GITHUB y como proveedor del servicio de email, utilizaríamos 
PROTONMAIL.
